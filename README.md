## Development
|             | Cmd                        |
| ---         | ---                        |
| **Install** | `yarn i && yarn bootstrap` |
| **Preview** | `yarn watch`               |
| **Build**   | `yarn build`               |
| **Test**    | `yarn test`                |

###### Environments
* production
* development (default)

```
NODE_ENV=production yarn build
```
