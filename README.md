# pijma
Yet another user interface components library. Weird. Slow. Our own.

## env

`MODE=production|development` - режим сборки (`developent` по-умолчанию)

## install

`yarn install`

## bootstrap

`lerna bootstrap`

## development

`yarn watch`

## build

`yarn build`

## demo

`yarn start` and open [http://localhost:8080](http://localhost:8080)
