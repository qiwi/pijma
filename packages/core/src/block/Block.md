## Состояния блоков

```jsx
<Block>
  <Box p={6}>
    <Typo as="p" display="block" size="4" height="6" weight="300" color="#000">
      default
    </Typo>
  </Box>
</Block>
```

```jsx
<Block active>
  <Box p={6}>
    <Typo as="p" display="block" size="4" height="6" weight="300" color="#000">
      active
    </Typo>
  </Box>
</Block>
```

```jsx
<Block active accent>
  <Box p={6}>
    <Typo as="p" display="block" size="4" height="6" weight="300" color="#000">
      active accent
    </Typo>
  </Box>
</Block>
```
