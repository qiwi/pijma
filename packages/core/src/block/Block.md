## Состояния блоков

```jsx
<Block>
  <Box p={6}>
    <Typo as="p" display="block" size="4" height="6" weight="300" color="#000">default</Typo>
  </Box>
</Block>
 ```

```jsx
<Block hover>
  <Box p={6}>
    <Typo as="p" display="block" size="4" height="6" weight="300" color="#000">hover</Typo>
  </Box>
</Block>
```

```jsx
<Block hover accent>
  <Box p={6}>
    <Typo as="p" display="block" size="4" height="6" weight="300" color="#000">hover accent</Typo>
  </Box>
</Block>
```
