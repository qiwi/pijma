## Расположение
Кнопки могут располагаться только вертикально.

```jsx
<Box>
  <Actions size="accent">
    <Button
      kind="brand"
      size="accent"
      text="Кнопка"
    />
    <Button
      kind="simple"
      size="accent"
      text="Кнопка"
    />
  </Actions>
</Box>
```


## Размеры
В зависимости от размера кнопок прописывается размер разделителей

#### Accent
```jsx
<Box>
  <Actions size="accent">
    <Button
      kind="brand"
      size="accent"
      text="Кнопка"
    />
    <Button
      kind="simple"
      size="accent"
      text="Кнопка"
    />
  </Actions>
</Box>
```

#### Normal
```jsx
<Box>
  <Actions size="normal">

      kind="brand"

```
