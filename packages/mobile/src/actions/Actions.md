## Расположение

Кнопки могут располагаться только вертикально.

```jsx
<Block>
  <BlockContent>
    <Actions size="accent">
      <Button
        kind="brand"
        size="accent"
        text="Кнопка"
      />
      <Button
        kind="simple"
        size="accent"
        text="Кнопка"
      />
    </Actions>
  </BlockContent>
</Block>    
```

## Размеры

В зависимости от размера кнопок прописывается размер разделителей

#### Accent

```jsx
<Block>
  <BlockContent>
    <Actions size="accent">
      <Button
        kind="brand"
        size="accent"
        text="Кнопка"
      />
      <Button
        kind="simple"
        size="accent"
        text="Кнопка"
      />
    </Actions>
  </BlockContent>
</Block>    
```
