## Виды аннотаций
```jsx
<Annotation children={{
  title: 'Условия',
  content: 'Подключить и расширить тариф можно в любое время',
  items: [
    {
      itemTitle: 'Что входит в тариф',
      itemContent: 'Переводы на любые карты Переводы через Систему быстрых платежей Погашение кредита',
    },
    {
      itemTitle: 'Сколько действует',
      itemContent: 'До конца текущего месяца. Остаток на следующий месяц не переносится',
    }
  ]
}}
/>
```
